# DIGITAL-NOMADS
A modern, responsive web platform designed for digital nomads, helping them discover the best places to live, work, and travel based on cost, safety, internet speed, community, and lifestyle.

# About the Project

Digital Nomad is a hackathon project focused on empowering remote workers.
The app helps users:
    1. Find the best cities for remote work
    2. Connect with other digital nomads
    3. Share personal stories, travel experiences & work tips
    4. Explore global locations with key data like cost of living, internet speed, and lifestyle

# Key Features

    Global City Explorer – Discover top digital nomad destinations

    Nomad Profile System – Create a profile and connect with others

    Experience Sharing Wall – Users can post stories, photos, or tips

    Remote Connections – Chat, network, and collaborate with travelers worldwide

    Cost of Living Insights – Compare expenses for different cities

    Internet Speed Data – Check connectivity for remote work

    Coworking Map – Find cafés and coworking spaces
